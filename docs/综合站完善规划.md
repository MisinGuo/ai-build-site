## 🎯 综合主站核心模块

### 模块总览

```
综合主站 (www.gamebox.com)
│
├── 1. 游戏库模块 (/games) ✅ 已有
├── 2. 游戏盒子模块 (/boxes) ✅ 已有
├── 3. 内容中心 (/content) ✅ 已实施
├── 4. 礼包中心 (/gifts) 🔧 待实施
├── 5. 搜索模块 (/search) 🆕 待实施
├── 6. 榜单模块 (/rankings) 🆕 待实施
├── 7. 用户中心 (/user) 🆕 待实施
└── 8. 专题活动 (/special) 🆕 待实施
```

### 1. 游戏库模块 ✅

**路由结构:**
```
/games                  # 游戏列表
/games/category/{slug}  # 品类聚合页
/games/{id}             # 游戏详情
```

**核心功能:**
- 游戏列表展示
- 按品类筛选
- 游戏详情页(含下载、礼包、攻略)
- 同品类游戏推荐

### 2. 游戏盒子模块 ✅

**路由结构:**
```
/boxes          # 盒子列表
/boxes/{id}     # 盒子详情
/boxes/compare  # 盒子对比(可选)
```

**核心功能:**
- 盒子列表展示
- 盒子详情
- 包含游戏列表
- 优惠对比

### 3. 内容中心 ✅ 已实施

**路由结构:**
```
/content                # 内容列表
/content?section=guides # 按板块筛选
/content/{id}           # 内容详情
```

**核心功能:**
- 统一管理攻略、资讯、评测
- 板块化分类(11种板块类型)
- 多维度筛选和排序
- 卡片式展示

**板块类型:**

#### 攻略类 (Guides)
- `guide_beginner` - 新手攻略
- `guide_advanced` - 进阶攻略
- `guide_pvp` - PVP攻略
- `guide_lineup` - 阵容攻略
- `guide_development` - 养成攻略

#### 资讯类 (News)
- `news_update` - 版本更新
- `news_event` - 活动资讯
- `news_industry` - 行业动态

#### 评测类 (Reviews)
- `review` - 游戏评测
- `comparison` - 横向对比

#### 专题类 (Topics)
- `topic` - 专题合集

**已创建文件:**
- ✅ `/src/config/pages/content.ts` - 配置文件
- ✅ `/src/app/[locale]/content/page.tsx` - 列表页
- ✅ `/src/app/[locale]/content/[slug]/page.tsx` - 详情页
- ✅ `/src/lib/api-types.ts` - API类型(添加section参数)
- ✅ `/src/lib/api.ts` - API方法
- ✅ `/docs/内容中心实施文档.md` - 实施指南

## 🎁 礼包模块规划

### 设计理念: "独立中心 + 场景化嵌入"

**优势:**
- ✅ 独立SEO入口 - 可做礼包相关关键词
- ✅ 多场景展示 - 在游戏详情、品类页等位置嵌入
- ✅ 用户体验好 - 一站式查看和领取
- ✅ 灵活运营 - 可做礼包活动和专题

### 路由结构

```
/gifts                      # 礼包中心主页
/gifts?category={slug}      # 按品类筛选
/gifts?game={id}            # 按游戏筛选
/gifts?type=hot             # 热门礼包
/gifts?type=expiring        # 即将过期
```

### 展示场景

#### 1. 独立礼包中心 (`/gifts`)

```tsx
<GiftCenterPage>
  {/* 顶部筛选 */}
  <GiftFilters>
    <Tabs>
      <Tab>全部礼包</Tab>
      <Tab>热门礼包</Tab>
      <Tab>新游礼包</Tab>
      <Tab>即将过期</Tab>
    </Tabs>
    <CategoryFilter /> {/* 品类筛选 */}
  </GiftFilters>

  {/* 礼包列表 */}
  <GiftGrid>
    <GiftCard
      gameName="XX仙侠"
      giftContent="钻石x1000, 金币x50000"
      validUntil="2026-02-28"
      remainCount={150}
      totalCount={500}
      status="可领取"
    />
  </GiftGrid>
</GiftCenterPage>
```

#### 2. 游戏详情页嵌入 (`/games/{id}`)

```tsx
<GameDetailPage>
  <GameInfo />
  <GameDownload />
  
  {/* 嵌入该游戏的礼包 */}
  <GameGiftsSection>
    <h2>专属礼包</h2>
    <GiftList gameId={id} inline />
    <Link href="/gifts?game={id}">查看更多礼包 →</Link>
  </GameGiftsSection>
  
  <RelatedGuides />
</GameDetailPage>
```

#### 3. 品类页嵌入 (`/games/category/{slug}`)

```tsx

  <CategoryIntro />
  <TopGames />
  
  {/* 品类礼包专区 */}

    <Link href="/gifts?category={slug}">
      查看所有{category.name}礼包 →
    </Link>
  </CategoryGiftsSection>
  
  <CategoryGuides />

```

#### 4. 首页展示 (`/`)

```tsx

  <HotGames />

  <HotGiftsSection>
```

## 💾 数据库设计

### 核心表结构

#### 1. 内容表 (已更新)

```sql
-- 攻略/内容表
CREATE TABLE gb_strategy_article (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  title VARCHAR(200) NOT NULL COMMENT '标题',
  content TEXT COMMENT '内容',
  summary TEXT COMMENT '摘要',
  cover_image VARCHAR(500) COMMENT '封面图',
  
  -- ⭐板块分类
  section VARCHAR(50) COMMENT '板块类型: guide_beginner, news_update等',
  
  -- 关联
  category_id BIGINT COMMENT '品类ID',
  game_id BIGINT COMMENT '游戏ID(可选)',
  
  -- 标签
  tags JSON COMMENT '标签数组',
  
  -- 统计
  view_count INT DEFAULT 0 COMMENT '浏览量',
  like_count INT DEFAULT 0 COMMENT '点赞数',
  
  -- 多语言
  locale VARCHAR(10) DEFAULT 'zh-CN' COMMENT '语言',
  
  -- 时间
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME ON UPDATE CURRENT_TIMESTAMP,
  
  -- 状态
  status TINYINT DEFAULT 1 COMMENT '状态: 0=草稿 1=已发布',
  del_flag TINYINT DEFAULT 0,
  
  INDEX idx_section (section),
  INDEX idx_category (category_id),
  INDEX idx_game (game_id),
  INDEX idx_locale (locale),
  INDEX idx_create_time (create_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='内容文章表';
```

#### 2. 礼包表

```sql
CREATE TABLE gb_gift (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100) NOT NULL COMMENT '礼包名称',
  game_id BIGINT NOT NULL COMMENT '游戏ID',
  
  -- 礼包内容
  content TEXT COMMENT '礼包内容描述',
  code VARCHAR(100) COMMENT '礼包码(领取后生成)',
  
  -- 数量控制
  total_count INT DEFAULT 0 COMMENT '总数量',
  received_count INT DEFAULT 0 COMMENT '已领取数量',
  
  -- 有效期
  valid_from DATETIME COMMENT '生效时间',
  valid_until DATETIME COMMENT '失效时间',
  
  -- 展示控制
  is_hot TINYINT DEFAULT 0 COMMENT '是否热门',
  is_new TINYINT DEFAULT 0 COMMENT '是否新礼包',
  sort_order INT DEFAULT 0 COMMENT '排序',
  
  -- 关联
  category_id BIGINT COMMENT '品类ID(冗余,方便查询)',
  
  -- 统计
  view_count INT DEFAULT 0 COMMENT '浏览量',
  
  -- 时间
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME ON UPDATE CURRENT_TIMESTAMP,
  
  -- 状态
  status TINYINT DEFAULT 1 COMMENT '状态: 0=下线 1=上线',
  del_flag TINYINT DEFAULT 0,
  
  INDEX idx_game (game_id),
  INDEX idx_category (category_id),
  INDEX idx_hot (is_hot),
  INDEX idx_valid (valid_from, valid_until),
  INDEX idx_create_time (create_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='礼包表';
```

#### 3. 用户收藏表

```sql
CREATE TABLE gb_user_favorite (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL COMMENT '用户ID',
  
  -- 收藏目标
  target_type VARCHAR(20) NOT NULL COMMENT '类型: game, box, article, gift',
  target_id BIGINT NOT NULL COMMENT '目标ID',
  
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  
  UNIQUE KEY uk_user_target (user_id, target_type, target_id),
  INDEX idx_user (user_id),
  INDEX idx_target (target_type, target_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户收藏表';
```

#### 4. 用户领取记录表

```sql
CREATE TABLE gb_user_gift_record (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL COMMENT '用户ID',
  gift_id BIGINT NOT NULL COMMENT '礼包ID',
  game_id BIGINT NOT NULL COMMENT '游戏ID',
  
  -- 礼包码
  gift_code VARCHAR(100) COMMENT '领取到的礼包码',
  
  -- 时间
  receive_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '领取时间',
  
  -- 状态
  is_used TINYINT DEFAULT 0 COMMENT '是否已使用',
  use_time DATETIME COMMENT '使用时间',
  
  INDEX idx_user (user_id),
  INDEX idx_gift (gift_id),
  INDEX idx_receive_time (receive_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户礼包领取记录';
```

#### 5. 专题表

```sql
CREATE TABLE gb_topic (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  title VARCHAR(200) NOT NULL COMMENT '专题标题',
  slug VARCHAR(100) UNIQUE COMMENT 'URL别名',
  description TEXT COMMENT '专题描述',
  
  -- 图片
  banner_image VARCHAR(500) COMMENT 'Banner图',
  cover_image VARCHAR(500) COMMENT '封面图',
  
  -- 内容关联(JSON数组)
  game_ids JSON COMMENT '关联游戏ID列表',
  article_ids JSON COMMENT '关联文章ID列表',
  gift_ids JSON COMMENT '关联礼包ID列表',
  
  -- 展示控制
  is_featured TINYINT DEFAULT 0 COMMENT '是否精选',
  sort_order INT DEFAULT 0,
  
  -- 时间
  start_time DATETIME COMMENT '开始时间',
  end_time DATETIME COMMENT '结束时间',
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME ON UPDATE CURRENT_TIMESTAMP,
  
  -- 状态
  status TINYINT DEFAULT 1,
  del_flag TINYINT DEFAULT 0,
  
  INDEX idx_slug (slug),
  INDEX idx_featured (is_featured),
  INDEX idx_time (start_time, end_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='专题表';
```

## 🔌 API接口规范

### 内容接口 (已实施)

```
GET /api/public/strategy-articles
参数:
  - locale: string (语言)
  - section: string (板块类型, 可选)
  - categoryId: number (品类ID, 可选)
  - pageNum: number
  - pageSize: number

GET /api/public/strategy-articles/{id}
参数:
  - locale: string (语言)
```

### 礼包接口 (待实施)

```
GET /api/public/gifts
参数:
  - locale: string (语言)
  - categoryId: number (品类筛选, 可选)
  - gameId: number (游戏筛选, 可选)
  - type: string (hot/new/expiring, 可选)
  - pageNum: number
  - pageSize: number

GET /api/public/gifts/{id}
参数:
  - locale: string

POST /api/user/gifts/claim
参数:
  - giftId: number
返回: { code, msg, data: { giftCode: string } }

GET /api/user/gifts/my-gifts
参数:
  - pageNum: number
  - pageSize: number
```

### 搜索接口

```
GET /api/public/search
参数:
  - q: string (搜索关键词)
  - type: string (games/boxes/content/gifts/all)
  - locale: string
  - pageNum: number
  - pageSize: number
```

### 榜单接口

```
GET /api/public/rankings
参数:
  - type: string (games/gifts/content)
  - category: string (品类slug, 可选)
  - period: string (week/month/all)
  - limit: number (默认20)
```
